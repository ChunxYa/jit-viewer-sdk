快速开始

直接在 HTML 中引入 SDK：

html 
<link rel="stylesheet" href="https://unpkg.com/jit-viewer@1.4.1/dist/iife/jit-viewer.min.css">
<script src="https://unpkg.com/jit-viewer@1.4.1/dist/iife/jit-viewer.min.js"></script>

<div id="viewer"></div>

<script>
  const viewer = JitViewer.createViewer({
    target: '#viewer',
    file: 'document.pdf',
    toolbar: true
  });

  viewer.mount();
</script>

或使用 npm 安装：

bash 
npm install jit-viewer@1.4.1

在 HTML 中直接引入（推荐）

通过 script 标签直接引入 IIFE 版本，无需构建工具：

html 
<!-- 引入 SDK 样式和脚本 -->
<link rel="stylesheet" href="https://unpkg.com/jit-viewer@1.4.1/dist/iife/jit-viewer.min.css">
<script src="https://unpkg.com/jit-viewer@1.4.1/dist/iife/jit-viewer.min.js"></script>

<!-- 创建预览容器 -->
<div id="viewer"></div>

<script>
  // 使用 createViewer API 创建预览器
  const viewer = JitViewer.createViewer({
    file: 'document.pdf',     // 文件URL或ArrayBuffer
    filename: 'document.pdf', // 文件名
    toolbar: true,            // 显示工具栏
    width: '100%',
    height: '600px',
    onReady: () => console.log('准备就绪'),
    onLoad: () => console.log('加载完成'),
    onError: (err) => console.error('错误:', err)
  });
  
  // 挂载到DOM
  viewer.mount('#viewer');
</script>

createViewer API

用于在 HTML/JS 中直接调用，无需 Vue 框架：

javascript 
// 通过 JitViewer 全局对象访问
const viewer = JitViewer.createViewer({
  file: 'document.pdf',      // 文件URL或ArrayBuffer
  filename: 'document.pdf',  // 可选：文件名
  toolbar: true,             // 显示工具栏
  theme: 'light',            // 主题：'light' | 'dark'
  width: '100%',
  height: '600px',
  cad: undefined,            // 可选：CAD 预览配置
  model3d: undefined,        // 可选：3D 模型预览配置
  onReady: () => console.log('准备就绪'),
  onLoad: () => console.log('加载完成'),
  onError: (err) => console.error('错误:', err)
})

// 挂载到 DOM
viewer.mount('#viewer-container')

// 可用方法
viewer.zoom(1.5)         // 设置缩放比例
viewer.rotate(90)       // 设置旋转角度
viewer.fullscreen(true) // 切换全屏
viewer.print()          // 打印文档
viewer.download()       // 下载文档
viewer.resetModel3DCamera() // 重置 3D 视角
viewer.setModel3DAutoRotate(true) // 控制 3D 自动旋转
viewer.toggleModel3DWireframe() // 切换 3D 线框
viewer.destroy()        // 销毁实例

CAD 说明：DXF 可直接预览，DWG / DWF 建议通过 cad.previewUrl 或 cad.adapter 接入转换结果。

3D 说明：当前首批支持 GLB / glTF / STL，也可以通过 model3d.previewUrl 或 model3d.adapter 接入转换后的模型资源。

ViewerInstance 方法

方法 参数 返回值 说明
mount HTMLElement | string Promise<void> 挂载到指定元素
destroy - void 销毁实例
setFile FileSource, string? Promise<void> 设置文件
zoom number void 设置缩放比例
rotate number void 设置旋转角度
fullscreen boolean void 切换全屏
print - Promise<void> 打印文档
download - Promise<void> 下载文档

ToolbarConfig 配置

typescript 
interface ToolbarConfig {
  items?: string[]           // 显示的工具项
  position?: 'top' | 'bottom' | 'both'  // 位置
  visible?: boolean          // 是否可见
}

// 默认工具项
const defaultItems = [
  'zoom',      // 缩放控制
  'rotate',    // 旋转控制
  'fullscreen', // 全屏
  'pagination', // 分页
  'print',     // 打印
  'download'   // 下载
]
